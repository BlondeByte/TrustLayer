## What is TrustLayer?

TrustLayer is a multi-agent AI system that analyzes content across 
two critical dimensions:

- **Authenticity** — Was this written by a human or generated by AI?
- **Credibility** — Can this content actually be trusted?

Built for security teams, content platforms, and businesses that need 
to know what they're working with before they act on it.

---

## How It Works

TrustLayer uses a sequential multi-agent architecture powered by the 
Anthropic API. Each agent specializes in one dimension of analysis 
and passes enriched findings to the next.
```
Input (text or URL)
        ↓
Orchestrator Agent
        ↓
[AUTHENTICITY CHAIN]        [CREDIBILITY CHAIN]
Linguistic Agent            Citation Extraction Agent
        ↓                           ↓
Behavioral Agent            Web Fetch Agent
                                    ↓
                            Relevance Agent
                                    ↓
                            Consistency Agent
                                    ↓
                            Confidence Language Agent
        ↓                           ↓
────────────────────────────────────────────────
                Report Synthesizer
                        ↓
            trustlayer_report_TIMESTAMP.md
```

---

## Agent Breakdown

| Agent | Role |
|-------|------|
| Orchestrator | Coordinates mode and prepares context |
| Linguistic | Analyzes sentence structure, vocabulary, voice |
| Behavioral | Analyzes information density, curiosity, opinion integrity |
| Citation Extraction | Finds all claims and URLs in the content |
| Web Fetch | Retrieves and reads cited URLs |
| Relevance | Determines if sources actually support claims |
| Consistency | Checks internal logical coherence |
| Confidence Language | Flags overclaiming and manipulation patterns |
| Synthesizer | Generates branded markdown report |

---

## Analysis Modes

| Mode | What it does |
|------|-------------|
| **Authenticity** | Detects human vs AI origin |
| **Credibility** | Verifies claims, sources, and consistency |
| **Full Analysis** | Runs both chains, unified report |

## Installation
```bash
git clone https://github.com/BlondeByte/TrustLayer.git
cd TrustLayer
pip3 install -r requirements.txt
```

Create a `.env` file in the root:
```
ANTHROPIC_API_KEY=your_api_key_here
```

---

## Usage
```bash
python3 main.py
```

Select your mode, paste text or submit a URL, and TrustLayer 
will run the full agent chain and generate a timestamped report.

## Disclaimer

TrustLayer assesses credibility signals — not ground truth.
A high score means content follows credible practices.
It does not guarantee factual accuracy.
For critical decisions, always verify primary sources directly.
